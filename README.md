# Python Bootcamp Projects

A collection of projects completed as part of a Python bootcamp / 100 Days of Code curriculum — covering core Python fundamentals, scripting, and small applications before moving into specialized work (e.g. computer vision, see `100-days-cv` repo).

## Structure

```
python-bootcamp-projects/
└── projects/
    ├── 01-project-name/
    ├── 02-project-name/
    └── ...
```

Each project folder will have its own short README describing what it does and how to run it.

## Setup

```bash
python -m venv venv
source venv/bin/activate   # or venv\Scripts\activate on Windows
pip install -r requirements.txt   # if applicable, per-project
```
